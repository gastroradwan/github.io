# GASTRORADWAN Web

Nový veřejný web společnosti GASTRORADWAN s.r.o.

## Účel webu

Web `gastroradwan.eu` bude sloužit jako veřejná internetová prezentace firmy a vstupní brána pro poptávky služeb.

## Směr prezentace

Hlavní sdělení:

**Revize jsou jen začátek.**

GASTRORADWAN se bude prezentovat jako partner pro komplexní správu elektro bezpečnosti firem:

- revize elektrických zařízení,
- revize strojů a technologií,
- revize hromosvodů,
- kontroly spotřebičů,
- školení dle NV 194/2022 Sb.,
- technická dokumentace,
- harmonogramy revizí,
- příprava na OIP, TIČR, audity a pojišťovny,
- budoucí zákaznický portál EZApp.

## Rozdělení projektů

- `gastroradwan/github.io` – veřejný web GASTRORADWAN.eu
- `gastroradwan/EZApp` – interní systém, databáze, zákaznický portál a zpracování zakázek

## Stav

Čistý nový začátek.
