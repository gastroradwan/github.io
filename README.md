# GASTRORADWAN Web

Veřejná internetová prezentace společnosti GASTRORADWAN s.r.o.

## Stav

Pracovní verze hlavní stránky podle schváleného směru:

- průmyslový tmavě modro-oranžový styl,
- hlavní sdělení „Revize jsou jen začátek“,
- služby: revize, školení, dokumentace, harmonogramy, OIP/TIČR,
- sekce EZApp,
- nezávazná poptávka.

## Rozdělení projektu

- `gastroradwan/github.io` – veřejný web GASTRORADWAN.eu
- `gastroradwan/EZApp` – interní systém, databáze, zákaznický portál a zpracování zakázek

## Další kroky

1. Zkontrolovat vzhled homepage.
2. Doplnit finální kontaktní údaje.
3. Připravit skutečný poptávkový formulář.
4. Napojit formulář na EZApp.
5. Přidat samostatné stránky služeb.
